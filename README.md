# Credit Scoring Model

This project predicts whether a customer is a good or bad credit risk using Machine Learning.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn

## Model Used

* Random Forest Classifier

## Features

* Income
* Age
* Loan Amount
* Credit History
* Employment Years
* Marital Status

## Output

* Accuracy Score
* Confusion Matrix
* Feature Importance Graph

## How to Run

1. Install required libraries
2. Run the Python file
3. View output and graphs
